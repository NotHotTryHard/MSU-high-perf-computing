# Генерация графика скейлинга для MPI+OpenMP

## Использование

Для генерации графика ускорения для сетки 1200×800:

```bash
cd "3. MPI + OpenMP"
source ../.venv/bin/activate
python3 plot_scaling_1200x800.py
```

Это создаст файл `3_scaling_1200x800.png` с графиком ускорения.

## Данные

- 4 MPI-процесса (фиксировано)
- Число потоков на процесс: 1, 2, 4, 8
- Времена: 11.84, 7.28, 6.12, 4.89 секунд
- Ускорения: 1.00×, 1.63×, 1.93×, 2.42×
